## Run Locally

Clone the project

```bash
  git clone https://github.com/David93400/DavidSoucany_7_18052022.git
```

Go to the project directory

```bash
  cd DavidSoucany_7_18052022
```

Install dependencies

```bash
  npm install
```
Build app

```bash
  npm run build
```

Start the server

```bash
  npm run start
```
